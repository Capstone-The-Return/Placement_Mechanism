Αυτό το script κάνει τα εξής:

Ορίζει τις απαιτήσεις των 3 υπηρεσιών σου (με τα νούμερα που δώσαμε).

Ορίζει 3 εικονικούς Servers (υποσύνολα του PC σου) για να έχει νόημα η επιλογή.

Υπολογίζει την ομοιότητα (Cosine Similarity) και διαλέγει τον καλύτερο server.

Αφαιρεί τους πόρους από τον server που επιλέχθηκε, ώστε να ενημερωθεί η διαθεσιμότητα για την επόμενη υπηρεσία (όπως σε ένα πραγματικό cloud).
